Introducing Atlas

See the plugin in action

It observes. It learns.
It becomes part of your world.

We didn't just write a plugin. We created a companion. Atlas resides locally on your server, listening to the heartbeat of your community. It doesn't just execute commands; it understands context.

Privacy First

Zero External Traffic.

Atlas utilizes no external LLM APIs. No OpenAI, no HTTP requests, no latency. It runs entirely on your hardware, ensuring your chat logs never leave your machine.

100% Local

Java 21.

Built on the bleeding edge.

Despite running on a 1.12.2 Paper server, Atlas leverages the sheer computational power of the Java 21 runtime to process conversation heuristics in real-time.

Adaptive Learning

Atlas watches chat patterns. The more your players talk, the smarter it gets at predicting needs.

Tangible Rewards

It doesn't just talk. Atlas can autonomously grant items based on the context of the struggle.

Fluent in Minecraft.

<Steve> Does anyone have food? I'm starving at spawn.
<Alex> I'm too far away, sorry.
[Atlas] I've detected you are at critical hunger levels, Steve.
* Atlas gave 16x Cooked Beef to Steve * [Atlas] Check your inventory. Stay safe.

* Simulated interaction based on v1.0 heuristics engine.

Technical Specifications

Core Requirement

Java 21 Runtime

While the server runs 1.12.2, the JVM must be updated to Java 21 to support Atlas's advanced data structures.

Platform

Paper 1.12.2

Optimized specifically for PaperSpigot. Spigot supported, but Paper is recommended for async processing.

Data Storage

SQLite / Flatfile

Zero database setup required. Atlas creates a local neural map of conversations in the plugin folder.

Networking

Offline Mode

Atlas makes 0 outgoing HTTP requests. It is safe for firewall-restricted environments.
